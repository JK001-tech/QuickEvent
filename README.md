Project Overview :

QuickEvent is a Full-Stack Event Management System built with Java, Spring Boot, Spring Security, Thymeleaf, Tailwind CSS, and MySQL.

This project demonstrates : Secure user authentication & role-based access control | CRUD operations for event management | Responsive UI using Thymeleaf + Tailwind CSS | MVC architecture & service-layer abstraction | Real-time event tracking and alerts.
